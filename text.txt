## 2) Environment Flag Audit
**Flags / Secrets**
- `VITE_BYPASS_PAYMENT` (client)
  - **Read in:** PurchasePreview.tsx, MarketplacePage.tsx, VirtualizedPixelGrid.tsx
  - **Effect:** Skips Razorpay, calls bypass Edge Functions directly; shows test badge.

- `BYPASS_PAYMENT_ENABLED` (server)
  - **Read in:** index.ts, index.ts
  - **Effect:** Allows bypass functions to execute and allocate assets without Razorpay.
  - **Risk:** If enabled in prod, any authenticated user can purchase without payment. No admin allowŌĆælist or environment allowŌĆælist.
- `VITE_RAZORPAY_KEY_ID` (client publishable)
  - **Read in:** PurchasePreview.tsx, MarketplacePage.tsx
  - **Effect:** Razorpay Checkout key.
  - **Risk:** Must be publishable; exposure is expected. Ensure it is not secret.
- `RAZORPAY_KEY_ID`, `RAZORPAY_KEY_SECRET` (server)
  - **Read in:** index.ts, index.ts, index.ts, index.ts
  - **Effect:** Order creation, payment verification, refunds.
  - **Risk:** Incorrect secret causes signature verification failures and false negatives; missing secret breaks payments.
- `SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY` (server)
  - **Read in:** All payment Edge Functions.
